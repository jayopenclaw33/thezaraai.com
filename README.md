# ZaraAI Website

This is the source for **thezaraai.com** — hosted via GitHub Pages.

## Pages
- `/index.html` — Homepage
- `/legal/privacy-policy.html` — Privacy Policy
- `/legal/terms.html` — Terms of Service

## Deploy via GitHub Pages
1. Push this folder to a GitHub repo named `thezaraai.com` (or any name)
2. Go to repo Settings → Pages → Source: main branch / root
3. Point your GoDaddy domain DNS to GitHub Pages
4. Done!
